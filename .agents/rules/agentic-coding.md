# Agentic Coding Guidelines

When working on this codebase:
- **Actively Explore**: Use your tools (`grep_search`, `list_dir`, `view_file`) to dynamically find code. Don't assume file structures.
- **Context Minimization**: Only view files that are strictly relevant to the current task to preserve token context.
- **Self-Verification**: Run tests (`npm run test:run` or workspace specific) and linters (`npm run lint`) frequently to verify your own changes.
